# YOUWARE.md

このファイルはYOUWARE Agent (youware.com) がこのリポジトリでコードを操作する際のガイダンスを提供します。

## プロジェクト概要

これはReact + TypeScript + Vite + Tailwind CSSを使用したシンプルなTODOリスト管理アプリケーションです。

## プロジェクト詳細

- **プロジェクト名**: TODOリスト管理アプリ
- **技術スタック**: React 18, TypeScript, Vite 7.0.0, Tailwind CSS 3.4.17, Zustand
- **エントリーポイント**: `src/main.tsx`
- **メインコンポーネント**: `src/components/TodoApp.tsx`

## 主要機能

1. **上段2枠レイアウト**:
   - 左上: 現在進行中のやることリスト（未完了タスク）
   - 右上: 完了タスクリスト（完了済みタスク）

2. **下段フォーム**:
   - やることの追加・編集用のテキストエリア
   - 選択中タスクと編集中タスクの状態表示

3. **ボタン群**:
   - 登録: 新しいタスクの追加
   - 編集: 選択したタスクの編集
   - 削除: 選択したタスクの削除
   - 更新: 編集中タスクの保存

4. **インタラクション**:
   - チェックボックス: タスクの完了/未完了切り替え
   - タスククリック: タスクの選択とフォームへのコピー
   - 完了タスクの自動移動

## アーキテクチャ

### 状態管理
- **Zustand** を使用してグローバル状態を管理
- `src/store/todoStore.ts`: メインストア
- `src/types/todo.ts`: TypeScript型定義

### コンポーネント構成
```
src/
├── App.tsx                 # アプリケーションルート
├── main.tsx               # エントリーポイント
├── components/
│   └── TodoApp.tsx        # メインTODOコンポーネント
├── store/
│   └── todoStore.ts       # Zustand状態管理
└── types/
    └── todo.ts            # 型定義
```

### データ構造
```typescript
interface Todo {
  id: string;
  text: string;
  completed: boolean;
  createdAt: Date;
}
```

## 開発コマンド

- **依存関係のインストール**: `npm install`
- **プロジェクトのビルド**: `npm run build`
- **開発サーバー**: `npm run dev` (開発時のみ使用)

## UI/UXの特徴

- **レスポンシブデザイン**: モバイル対応のグリッドレイアウト
- **視覚的フィードバック**: 選択状態のハイライト表示
- **アクセシビリティ**: 適切なARIA属性とキーボードナビゲーション
- **状態表示**: 現在の操作状態（選択中・編集中）を明確に表示

## 技術仕様

- **React Hooks**: 関数コンポーネントとHooksベースの実装
- **TypeScript**: 完全な型安全性
- **Tailwind CSS**: ユーティリティファーストのCSS
- **状態管理**: Zustandによる軽量な状態管理
